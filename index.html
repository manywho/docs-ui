Getting Started with custom components and debugging

A custom component demo

To run this custom component example locally you can clone this repository:
https://github.com/manywho/docs-ui/

You will need to run an http server locally to serve the player and custom component JavaScript file. The http-server-spa npm package is ideal for this. To install it globally on your computer (node required), run:
npm install http-server-spa -g

Navigate to the custom-input directory using a terminal and run:
http-server-spa . index.html 8080

Flow will now be accessible at http://localhost:8080, however you will need to add the same path and parameters as required on https://flow.boomi.com to run a flow (flow-id, flow-version-id).

This will load a flow that uses our custom player and imports our custom-input.js component.

  • Try entering some text into the input.

  • Try editing the custom-input.js file and refresh the page.

  • Try inspecting the styling in your browser developer tools and then adding some more CSS overrides to the index.html file.
